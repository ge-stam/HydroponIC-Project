\doxysubsection{Detailed Description}
This is part of Adafruit\textquotesingle{}s Neo\+Pixel library for the Arduino platform, allowing a broad range of microcontroller boards (most AVR boards, many ARM devices, ESP8266 and ESP32, among others) to control Adafruit Neo\+Pixels, FLORA RGB Smart Pixels and compatible devices -- WS2811, WS2812, WS2812B, SK6812, etc.
